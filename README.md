# 西安交通信息开发框架

# 模块划分
```
01-xci-module(模块)
	xci-core(核心模块)
	xci-upload(文件上传模块)
	xci-sys(系统模块)
02-xci-starters(spring-boot-starters自动装配)
	xci-api-spring-boot-starter(api接口自动装配)
	xci-web-spring-boot-starter(web网页自动装配)
	xci-upload-spring-boot-starter(文件上传模块自动装配)
	xci-sys-spring-boot-starter(系统模块自动装配)
03-xci-ui(UI)
	xci-jxui
04-xci-demo(测试应用)
	xci-demo-api
	xci-demo-web
	xci-demo-upload
	xci-demo-sys
05-xci-doc(文档)
    sql
```
